## 通过一个故事理解原型链
你，你爸，你爷爷 去买烟，售货员跟你要5块钱。你没有，你就看了你爸一眼，你爸摸摸兜，没有，看了你爷爷一眼，你爷爷摸摸兜，找到五块钱，给了售货员，你拿走了烟；
另一种情况就是，你爷爷也没有，然后爷仨被集体鄙视了，穷逼。
